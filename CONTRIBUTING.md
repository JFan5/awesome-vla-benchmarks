# Contributing

Contributions are welcome! Please follow these guidelines:

## Adding a Benchmark

1. Fork this repository
2. Add the benchmark to the appropriate table in `README.md`
3. Include: name, year, simulator/embodiment, task count, key focus, and links (paper + code)
4. Submit a pull request

## Requirements

- The benchmark must be publicly available (paper published or preprint on arXiv)
- Code or dataset should be accessible (or announced for release)
- The benchmark should be relevant to evaluating VLA models or closely related robot learning paradigms

## Format

Follow the existing table format. Keep descriptions concise. Link to the official paper (preferably arXiv) and code repository.
